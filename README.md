# Contao Check-Loader

Der "Contao Check-Loader" ist für das CMS Contao bzw. für dessen Check-Tool. Mit dem Check-Tool kann eine vorhandene Contao-Installation geprüft oder aber auch eine Installation gestartet werden.

Ziel des "Contao Check-Loader" ist es, mit nur *einer, unveränderlichen Datei* die aktuelle Check-Version automatisch auf einen Server zu laden und zu starten.

Aktuell muss man die Check-Version aus dem Git-Account downloaden (auf den Server direkt, oder auf eigenen Rechner), entpacken, ggf. wieder per FTP uploaden und starten. Ist eine alte Version schon vorhanden, muss man diese vorher löschen.

Mit dem "Contao Check-Loader" muss man nur noch eine Datei in das Basisverzeichnis laden und aufrufen.

## Installation
Einfach die Datei "contao-checkloader.php" in das Basisverzeichnis der vorhandenen oder zukünftigen Contao-Installation hochladen (FTP).

## Ausführung
Die Domain + die Datei aufrufen z.B. http://www.meine-domain.de/contao-checkloader.php
